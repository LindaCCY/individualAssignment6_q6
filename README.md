# Polyline and Polygon Map App
An Android app that displays Google Maps with interactive polylines and polygons for hiking trails and parks.

## Features
Google Maps integration
Polyline: Represents a hiking trail route
Polygon: Highlights a park or area of interest
Customizable polyline color and width
Customizable polygon fill and stroke colors
Click listeners on overlays to show information
Interactive map controls

## How It Works
The app uses Google Maps SDK to:
Draw polylines connecting GPS coordinates (hiking trail)
Draw polygons to mark enclosed areas (parks)
Handle tap events on map overlays
Display details when polylines/polygons are clicked

## Functionality
Displays a map with predefined hiking trail (polyline)
Shows a highlighted park area (polygon)
Customize overlay appearance (colors, width)
Tap on trail or park to see information

## setup
Clone the repository
Add your Google Maps API key to local.properties:
Open in Android Studio
Build and run
